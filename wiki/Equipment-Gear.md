# Equipment & Gear System

## Gear Slots

Each hero has **12 equipment slots**:

| Slot | Type | Notes |
|------|------|-------|
| Main Hand | Weapon | Melee, Ranged, or Demolitions |
| Off Hand | Shield | Cannot use with two-handed weapons |
| Armor | Chest | Primary defensive slot |
| Legs | Lower body | Defense + HP |
| Gloves | Hands | Accuracy + Crit |
| Boots | Feet | Turn Speed + Movement |
| Ring 1 | Accessory | General ring |
| Ring 2 | Accessory | General ring |
| Ring 3 | Accessory | **Stat Focus Ring only** |
| Earring | Accessory | Status Resist focused |
| Necklace | Accessory | HP + Crit Damage |
