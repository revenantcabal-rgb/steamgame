## HOW SETS WORK

Green set items appear with a green border. Wearing multiple pieces from the same set on a single hero activates progressive bonuses. More pieces = stronger set bonus.

- Set bonuses are **additive** with gear stats
- You can mix set pieces with non-set pieces for partial bonuses
- Set pieces can still have facets, rarity bonuses, and enchantments
- Set bonuses apply per hero (not party-wide)
