# Wasteland Grind - Starlight Sphere Grid

The Starlight Sphere Grid is an account-wide passive progression system. Players spend Icqor Chess Pieces (dropped by high-level bosses) to unlock permanent bonuses across 7 themed paths with 15 nodes each (105 nodes total).

## SYSTEM OVERVIEW

- **7 paths**, each with **15 nodes** = **105 total nodes**
- Nodes are unlocked by spending **Icqor Chess Pieces**
- Bonuses are **account-wide** and apply to all heroes and all character slots
- Each path follows a linear prerequisite chain (node 1 must be unlocked before node 2, etc.)
- Nodes increase in cost deeper into the path
- Later nodes provide more powerful bonuses
- Progress is permanent and cannot be reset

---

## ICQOR CHESS PIECE DROPS

Icqor Chess Pieces drop from bosses in combat zones 4-7. Higher-level bosses have better drop rates and quantities.

| Boss | Zone | Min Level | Drop Chance | Quantity |
|------|------|-----------|-------------|----------|
| Raider Warlord | The Deadlands | 45+ | 15% | 1 |
| Commander Mech | Military Zone | 60+ | 30% | 1-2 |
| The Source | The Core | 80+ | 50% | 1-3 |
| The Cataclysm | Ground Zero | 95+ | 80% | 2-5 |

**Notes:**
- Drop chance rolls once per boss kill
- Higher difficulty tiers do not increase the base drop rate, but bosses are killed more frequently at higher tiers
- Icqor Chess Pieces are a tradeable resource on the Marketplace (sell value: 50 WC)

## PREREQUISITE SYSTEM

Each path is a **linear chain**. You must unlock nodes in order (1 -> 2 -> 3 -> ... -> 15).

**Cross-path requirements:** None. Each path is independent. You can invest in any combination of paths.

**Node cost scaling:**

| Node Position | Cost (Icqor Chess Pieces) |
|---------------|---------------------------|
| 1-3 | 1 each |
| 4-6 | 2 each |
| 7-9 | 3 each |
| 10-12 | 5 each |
| 13-14 | 8 each |
| 15 (Capstone) | 15 |

**Total cost per path:** 3 + 6 + 9 + 15 + 16 + 15 = **64 Icqor Chess Pieces**
**Total cost all paths:** 64 x 7 = **448 Icqor Chess Pieces**

## ACCOUNT-WIDE BONUSES

All bonuses from unlocked Starlight nodes apply to:
- Every hero on every character slot
- Both combat and specialist heroes
- Gathering and production bonuses stack with skill-based bonuses
- Percentage bonuses are additive with other percentage sources
